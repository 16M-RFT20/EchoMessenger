# (C# 코딩) 에코 메신저

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 사용자 키보드 입력을 받아서 처리하는 프로그램
- 사용한 플랫폼:
  - C#, .NET windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, TextBox, ListBox, Button
- 사용한 기술과 구현한 기능:
  - Visual Studio를 이용하여 UI 디자인
  - String 클래스를 이용한 사용자 입력 데이터 처리
  - DateTime 클래스를 이용한 현재시간 정보 구하기
  - Count 클래스를 이용하여 리스트에 쌓인 메시지 카운팅
  - Trim 클래스를 이용하여 메시지 앞뒤 불필요한 공백 제거
  - SelectedIndex 클래스를 이용하여 선택된 항목을 확인
  -  Length 클래스를 이용하여 메시지의 길이를 확인
