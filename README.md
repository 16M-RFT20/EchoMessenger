# (C# 코딩) 에코 메신저

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 사용자 키보드 입력을 받아서 처리하는 프로그램
- 사용한 플랫폼:
  - C#, .NET windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, TextBox, ListBox, Button
- 사용한 기술과 구현한 기능:
  - Visual Studio를 이용하여 UI 디자인
  - String 클래스를 이용한 사용자 입력 데이터 처리
  - DateTime 클래스를 이용한 현재시간 정보 구하기
  - Count 클래스를 이용하여 리스트에 쌓인 메시지 카운팅
  - Trim 클래스를 이용하여 메시지 앞뒤 불필요한 공백 제거
  - SelectedIndex 클래스를 이용하여 선택된 항목을 확인
  -  Length 클래스를 이용하여 메시지의 길이를 확인

## 실행 화면 (과제2)
- 과제2 코드의 실행 스크린샷

![과제2 실행화면](https://github.com/user-attachments/assets/c1e50b8c-959c-469c-8902-45a852670d79)


- 과제 내용
 - 입력창의 기존 메시지 지우기
 - 입력창에 입력 포커스 갖다 놓기
 - 엔터키로 전송하기
 - 입력 방어

- 구현 내용과 기능 설명
 - 전송이 끝나면 입력창에 남겨진 기존 메시지를 삭제합니다.
 - 전송 후에 마우스로 입력창을 다시 클릭하지 않아도 되도록 커서를 자동으로 입력창에 둡니다.
 - 마우스 클릭 대신 키보드의 Enter 키를 눌러도 메시지가 전송되도록 합니다.
 - 내용이 없는 빈 문자열이나 공백(Space)만 있을 때는 메시지가 전송되지 않도록 방지합니다.

![과제4 실행화면](https://github.com/user-attachments/assets/252af20d-e0cd-428f-b4f4-734f184046c1)



- 과제 내용
 - 선택 항목 제거
 - 전체 초기화
 - 글자 수 제한

- 구현 내용과 기능 설명
 - ListBox에서 특정 메시지를 마우스로 클릭하고 '삭제' 버튼을 누르면 해당 항목만 목록에서 제거합니다.
 - '전체 삭제' 버튼을 클릭하면 리스트의 모든 내용을 한번에 지웁니다.
 - 입력창에 글자 수를 50자로 제한하고, 초과시 사용자에게 경고 메시지를 띄웁니다.
